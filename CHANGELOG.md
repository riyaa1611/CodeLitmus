# Changelog

## [0.1.0] - 2026-04-17

### Added
- Initial release
- Workspace scanner with critical path detection (auth, payments, DB, API routes, security)
- AI-powered quiz generation via OpenRouter
- Adaptive difficulty (1–5)
- Per-file, per-category, and overall understanding scores
- Danger zone detection (high risk + low understanding)
- Quiz webview with instant feedback
- Report dashboard with SVG charts and progress timeline
- Sidebar tree views: Understanding Scores + Danger Zones
- CodeLens annotations on critical functions
- Status bar score indicator
- Inline diagnostics on danger zone lines
- Hover cards with audit status
- First-launch onboarding with API key setup
- Supports TypeScript, JavaScript, Python
